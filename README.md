# 🔄 Employee Lifecycle Framework

### By Claudia González Tornquist | HR Operations Leader | Global People Strategy

Over 20 years designing and running HR departments from scratch across multinationals, NGOs, startups, and consulting firms in the Americas and EMEA, I've built one core belief: **a great hire is only the beginning**. This repository documents the frameworks, playbooks, and operating systems I've built to take people from Day 1 through their full journey — onboarding, development, performance, culture, and beyond.

---

## 📌 What's in This Framework

- HR Operating Model — full functional coverage across 6 HR areas
- Onboarding system — preboarding through Day 90 integration
- Learning & Development framework — LMS deployment, leadership pipelines, certification programs
- Performance management — KPI frameworks, OKR design, feedback cycles
- Culture & Engagement — pulse surveys, recognition, employer brand
- Compensation benchmarking — global methodology using Mercer, WTW, Radford, BLS
- HR Operations & Compliance — payroll workflow, global labor law frameworks
- Operating cadence — daily through quarterly rhythms

## 🧭 The Employee Lifecycle

This framework covers every stage of the employee journey:

**Workforce Planning → Talent Acquisition → Onboarding → Learning & Development → Performance → Culture & Engagement → Compensation → Offboarding**

Each stage is documented with clear ownership, workflows, timelines, and measurable outcomes.

## 📊 Operating Cadence

| Frequency | Focus |
|---|---|
| Daily | Pipeline management, attendance, onboarding tracking |
| Weekly | HR Ops review, recruiting funnel, EOW reporting |
| Bi-Weekly | Payroll processing, dashboard updates |
| Monthly | Headcount review, L&D tracking, recognition |
| Quarterly | Pulse surveys, OKR review, compensation benchmarking, file audits |
